# FIFA Player Clustering using KMeans

## 📌 Business Problem
In football analytics, grouping players based on similar attributes can help in scouting, team building, and strategy planning. The goal of this project is to identify patterns among players using clustering techniques.

---

## 🎯 Project Objective
- Analyze FIFA player data
- Group similar players based on attributes
- Identify meaningful player segments using clustering

---

## 📊 Dataset
- FIFA player dataset
- Includes player attributes such as overall rating, pace, shooting, passing, dribbling, defending, and physical ability

---

## 🧠 Approach

### Data Preprocessing
- Cleaned the dataset
- Selected relevant numerical player attributes
- Prepared the data for clustering analysis

### Exploratory Data Analysis
- Performed EDA to understand distributions and relationships among player features

### Clustering
- Applied **KMeans clustering** to segment players into groups
- Used the **Elbow Method** to determine the optimal number of clusters

---

## 🤖 Techniques Used
- Exploratory Data Analysis (EDA)
- KMeans Clustering
- Elbow Method

---

## 📈 Key Insights
- Players can be grouped into distinct clusters based on performance and skill attributes
- Different clusters represent different player profiles and play styles
- Clustering helps in understanding player similarity beyond just overall rating

---

## 💡 Business Impact
- Helps in player scouting and recruitment
- Supports team strategy and squad planning
- Useful for identifying similar player profiles

---
